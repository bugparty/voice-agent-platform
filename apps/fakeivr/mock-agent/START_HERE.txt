This is a MOCK system that simulates an AI agent navigating phone menus:

  Real Phone System          →    Your Mock System
  ─────────────────────           ────────────────
  User calls pharmacy        →    Load JSON data
  Voice says "Press 1..."    →    Print IVR prompt
  User presses buttons       →    LLM decides option
  Navigate through menus     →    Navigate JSON tree
  Eventually reach human     →    Find TRANSFER_TO_HUMAN

Everything is TEXT-BASED in your terminal:
  • No actual phone call
  • No audio/speech
  • Just printed text showing the process
  • Perfect for testing and demonstration!

╔══════════════════════════════════════════════════════════════════════╗
║                    🎮 TRY IT NOW!                                    ║
╚══════════════════════════════════════════════════════════════════════╝

Run this command to see it in action:

    npm run simulate

Watch as the agent:
  1. "Calls" the pharmacy (loads JSON)
  2. Reads IVR prompts (from mock data)
  3. Decides which option to press (using LLM)
  4. Navigates through 6 menu levels
  5. Successfully connects to human!

The whole journey takes about 30-40 seconds.
